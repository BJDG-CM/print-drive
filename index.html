어떻게 파일을 열까요?

신뢰하는 기기에서는 전체 보관함을 열고, 공용 기기에서는 받은 링크의 안내만 확인하세요.

v2 공유 링크는 선택 파일 키만 전달합니다. 이전 #file=<id> 위치 링크는 전체 보관함 비밀번호가 필요합니다.

공유 링크 안내

공유 링크는 두 종류입니다

링크를 주소창에서 열면 Print Drive가 형식을 확인하고 가능한 가장 제한된 화면으로 안내합니다.

v2 파일별 capability 링크
  • 선택한 파일의 키만 임의 secret으로 보호해 전달하며 master key는 포함하지 않습니다.
  • secret은 URL fragment에서 읽은 즉시 주소창과 방문 기록의 현재 항목에서 제거합니다.
  • 링크 자체가 선택 파일의 접근 권한입니다. 비밀번호 없이 열리므로 신뢰하는 채널로만 전달하고, 전달 뒤에는 회수할 수 없다고 가정하세요.
  • 표시된 만료 시각은 이 브라우저가 확인합니다. 정적 호스팅은 만료·사용 횟수·회수를 서버에서 강제할 수 없습니다.
이전 #file=<id> 위치 링크
  • 파일 위치만 가리키며 파일별 key를 전달하지 않습니다.
  • 전체 보관함 비밀번호가 필요하므로 제한 공유나 만료 링크가 아닙니다.
  • 공용 기기에서는 세션 유지 옵션을 선택하지 마세요.

    공유 링크 QR

    v2 링크 자체가 선택 파일의 접근 권한이므로 신뢰 채널로만 전달하세요. QR은 링크 길이에 따라 만들지 못할 수 있으며 그때는 복사를 사용합니다. master key는 포함하지 않습니다. 이전 위치 링크 QR은 전체 보관함 비밀번호가 필요합니다. 정적 호스팅은 두 링크의 만료·사용 횟수를 서버에서 강제하지 못합니다.
